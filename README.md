# Gulp-Starter

## 使い方
For Dev -> npm run dev<br>
For Prod -> npm run prod<br>


###　機能
- sass
- PostCSS
- eslint
- babel
- image のサイズ調整
- HTML, CSS, js, image ファイルの rename
- html ファイルに css, js ファイルを自動挿入
- [production] - css, js, image ファイルの圧縮
- [production] - sourcemap の生成
- [development] - 開発用の Web サーバーを立てる(change で refresh)
